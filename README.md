# FILTRIS - Filipino Word Scramble Game

Created by:
Praswel and Group

How to Run the Game:

1. Download all files from the repository.

2. After downloading extract file.

3. Double click:
   FILTRIS_FINAL.exe

4. If Windows shows a security warning:

   * Click "More info"
   * Click "Run anyway"

Game Features:

* Filipino word scramble gameplay
* Multiple difficulty levels
* Clue system
* Score and health system
* Background image
* Game over sound
* Pause and navigation system

Controls:

* Type the correct word
* Press ENTER to submit answer
* Press ESC to pause the game

Important:
Do not separate the files from the folder or the image/sound may not work properly.
